## 목적
- GraphQL과 일렉트론을 공부하려고 제작했습니다.

## 소개 

### 방문한 숙소를 찍은 사진과 그 당시 느낀 감정을 한 줄로 작성하고, 업로드한 사진에 날짜와 위치정보가 있으면 자동으로 날짜와 구글맵으로 표기합니다. 오픈라인 상에서 일렉트론을 활용하여 추억을 남기기 위한 프로그램을 웹 페이지로 만들었습니다.
